# AI Radio Frontend Art Direction

## Product Positioning

This is NOT:

* a music player
* a retro radio
* a Spotify clone
* a chatbot UI
* a dashboard

This IS:

# A futuristic AI emotional audio interface.

The feeling should be:

* intelligent
* ambient
* minimal
* emotional
* alive
* reactive
* immersive

The user should feel like:

> they are connecting to an AI emotional frequency.

---

# Core Design Direction

Avoid:

* retro radio aesthetics
* vinyl players
* cassette tapes
* wooden textures
* skeuomorphic knobs
* overly colorful cyberpunk
* dashboard cards
* SaaS UI

Instead build:

# A Neural Audio System

Inspired by:

* Her
* Blade Runner 2049
* Ex Machina
* Linear
* Raycast
* Arc Browser
* futuristic operating systems
* signal processing interfaces
* AI neural systems

---

# Visual Philosophy

The interface should feel:

* calm
* futuristic
* responsive
* atmospheric
* intelligent
* cinematic

The UI should breathe slowly.

Everything should feel connected to sound and emotion.

---

#0B0B0C
#111111
#18181B
```

## Text Colors

```

---

## Accent Colors

Primary AI green:

```css
#7CFFB2
```

Secondary:

```css
#00FF85
#5EEAD4
```

Use accent colors sparingly.

Target ratio:

* 80% black
* 15% grayscale
* 5% green

# Overall UI Style

Minimal.
Clean.
Spatial.
Cinematic.

No clutter.

No card-heavy layouts.

No dashboard feeling.

Very large negative space.

---

# Core Interface Concept

The homepage should feel like:

# A live AI signal interface.

Not a landing page.

Not an app homepage.

---

# Main Layout Structure

## Layer 1 — Ambient Background

Subtle animated background:

* moving gradients
* floating particles
* soft noise grain
* signal scan lines
* atmospheric blur
* low opacity waveform textures

Very subtle.

---

## Layer 2 — Signal Grid

Very faint grid system.

Feels like a futuristic audio operating system.

---

## Layer 3 — Massive Reactive Waveform

The waveform is the main visual identity.

Not decorative.

It should react to:

* music
* AI voice
* user typing
* emotional intensity

Style:

* thin
* elegant
* glowing
* minimal

---

## Layer 4 — AI Core

Centerpiece of the interface.

A glowing AI signal orb / neural core.

This is the product memory point.

The orb should:

* pulse slowly
* react to sound
* glow softly
* distort subtly
* feel alive

---

## Layer 5 — Floating Player

Minimal floating player at the bottom.

Do not use traditional music player UI.

Keep it elegant and lightweight.

---

# Hero Section

Do NOT say:

* “Welcome”
* “AI Assistant”
* “Chat with AI”
* “Music Recommendation”

Instead use language like:

```txt
SIGNAL DETECTED

GPT Neural Radio

Listening to your emotional frequency.
```

or

```txt
CONNECTED

Emotional transmission active.
```

---

# Input Experience

Do NOT use a normal input field.

Avoid:

```txt
How are you feeling?
```

Instead create:

# A neural terminal input.

Examples:

```txt
> tell me how tonight feels_
```

or

```txt
What frequency are you on tonight?
```

The cursor should blink softly.

Waveforms should react while typing.

---

# Music Presentation

Do NOT display songs as playlist cards.

Do NOT look like Spotify.

Instead present tracks like signal transmissions.

Example:

```txt
NOW TRANSMITTING

ODESZA
A Moment Apart

signal stability: 92%
emotional resonance: high
```

The experience should feel like receiving emotional broadcasts.

---

# AI DJ Behavior

The AI is NOT a chatbot.

It is a live emotional signal entity.

It should speak minimally and poetically.

Examples:

```txt
“This one is for people pretending they’re okay tonight.”
```

or

```txt
“Signal quality unstable.
You sound emotionally overloaded.”
```

---

# Motion Design Principles

Everything must move slowly.

No fast UI motion.

No aggressive animations.

No gaming aesthetics.

Motion should feel like:

* breathing
* pulse
* electrical current
* signal drift
* ambient energy

---

# Animation Guidelines

Use:

* slow glow pulses
* waveform motion
* particle drift
* blur breathing
* soft reactive lighting

Avoid:

* excessive spring animations
* bouncing UI
* flashy transitions
* RGB effects

The entire interface should react to sound.

Examples:

* bass affects blur intensity
* vocals affect waveform motion
* AI voice affects core orb pulse
* emotional intensity affects ambient lighting

This is critical.

The interface must feel alive.
